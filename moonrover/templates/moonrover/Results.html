This challenge is an example of an optimal routing problem, also known as the travelling salesman problem. Companies face routing problems mainly in logistics, where an optimal route must be designed between nodes in a network such that cost is minimised.

Logistics companies such as Amazon have for long been planning to use drones as delivery vehicles, and the limited range of the drone poses a challenge for designing an optimal route for it. There are also ridesharing platforms in London and other large cities, where companies allow customers to rent vehicles and return them to selected locations, and these platforms face the travelling salesman problem when they design the network. Engineers use the network routing problem to design efficient ways to route information on microchips and fibre optic networks.

The problem is one of the most difficult ones to solve efficiently. Analytics can help in optimal routing problems by describing the problem as a network and using efficient algorithms to solve it.
