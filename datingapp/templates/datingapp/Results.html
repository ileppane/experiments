This challenge is an example of an assignment problem, a common problem class in combinatorial optimisation. Companies face assignment problems when they need to match workers with jobs, machines with orders, contracts with bidders, or drivers with passengers.

And of course, online dating apps develop sophisticated algorithms that match people together romantically. These algorithms are based on the assignment problem, and also utilise such techniques as machine learning to predict the success of the match.

Analytics is beneficial in the assignment problem, because we can describe the problem in a precise fashion and solve it using computational methods. Small-scale assignment problems can be solved using spreadsheet software (such as MS Excel), but sometimes there are thousands of possible pairings, which makes it computationally challenging to find feasible solutions.

National railway companies, for example, have hundreds of employees working in trains and it is challenging to find an optimal rostering given that the trains constantly change locations. Analytics can help with that.
